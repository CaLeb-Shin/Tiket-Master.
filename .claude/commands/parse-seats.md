# 좌석파싱 스킬

좌석배치도 엑셀(.xlsx/.xls)을 파싱하여 상품별좌석현황 엑셀을 생성합니다.

## 사용법

```
/parse-seats <엑셀파일경로> [출력파일경로]
```

## 실행 방법

1. 입력 파일 경로를 확인합니다
2. 다음 명령어를 실행합니다:

```bash
cd /Users/erwin_shin/Desktop/App./멜론티켓 && node scripts/seat-parser.js "$ARGUMENTS"
```

3. 결과를 확인하고 사용자에게 등급별 좌석 수를 보고합니다
4. 기존 타겟 파일이 있으면 compare-seats.js로 비교합니다:

```bash
node scripts/compare-seats.js <출력파일> <타겟파일>
```

5. 차이가 있으면 분석하고 파서를 수정합니다
6. 결과를 네이버 티켓 마스터플랜.md에 기록합니다

## 지원 형식 (3가지 자동 감지)

| 형식 | 헤더 패턴 | 예시 공연장 |
|------|----------|------------|
| 블록 | `X블록(N)` | 부산시민회관 |
| 열 | `X열 (N)` + 층 마커 | 대전예술의전당, 창원성산아트홀 |
| 구역 | `N층 X구역(N)` | 대구콘서트하우스 |

## 색상 → 등급 매핑

- **범례 우선**: 엑셀 오른쪽에 범례가 있으면 자동 사용
- **HSL 분류 (fallback)**: 빨강/핑크=VIP, 파랑=R, 초록=S, 노랑=A, 보라=시야방해R, 밝은초록=시야방해S, 검정=미판매
- **구역 형식 전용**: 핑크=VIP, 파랑=R, 보라=S, 주황=A, 무색=A석

## 출력 형식

상품별좌석현황 엑셀 (No | 이용일 | 회차 | 좌석등급 | 층 | 열 | 좌석수 | 좌석번호)

## 학습 이력

- 1→2: 블록 형식 100% ✅
- 5-1→5-2: 열 형식 100% ✅
- 6-1→6-2: 구역 형식 100% ✅
- 7-1→7-2: 열 형식 99.87% ✅
- 8-1→8-2: 파싱OK, 통합열 출력 미지원
