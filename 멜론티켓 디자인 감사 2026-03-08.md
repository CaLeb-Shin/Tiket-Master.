# 멜론티켓 디자인 감사 2026-03-08

작성일: 2026-03-08
범위: 사용자 앱, 공개 모바일 티켓, 어드민 주요 화면, 데스크톱 웹 공개 티켓

## 1. 총평

멜론티켓은 이미 `일반 예매 앱`처럼 보이지 않는 개성을 갖고 있다.
특히 공개 모바일 티켓은 버건디와 종이 질감, 전용 카드 구조 덕분에
방향 자체는 맞다.

문제는 현재 `고급스럽다`보다 `분열돼 보인다`, `어둡고 답답하다`,
`읽기 어렵다`, `눌러도 반응이 약하다` 쪽으로 체감된다는 점이다.

즉, 지금은 디자인 수준이 낮다기보다 `좋은 재료가 서로 안 맞물리는
상태`에 가깝다.

## 2. 현재 강점

### 2-1. 공개 티켓의 세계관은 이미 있다

- 공개 티켓은 일반 예매 앱 화면이 아니라 `별도 오브젝트`처럼 보인다.
- 버건디 계열, 플립 카드, 그룹 허브, QR 중심 구조는 인스타/카카오 공유용
  자산으로 발전시킬 수 있는 기반이 있다.

### 2-2. 메인 앱은 비교적 안정적이다

- 메인 앱은 아이보리 배경과 버건디 텍스트를 바탕으로 정보 UI가 깔끔하게
  정리돼 있다.
- 공연 상세, 결제, 내 티켓, 티켓 상세는 기본적인 정보 위계는 이미 갖춰져 있다.

### 2-3. 어드민은 운영도구로서 밀도는 충분하다

- 어드민은 한 화면에서 많은 상태를 처리할 수 있게 설계되어 있다.
- `네이버 주문 관리` 화면은 운영 툴로서의 방향이 분명하다.

## 3. 핵심 문제

### 3-1. 서비스가 하나로 안 보인다

- 사용자 앱은 `밝은 아이보리 + 버건디`
- 공개 티켓은 `어두운 버건디 + 블랙`
- 어드민은 `다크 차콜 + 웜 골드`

각 화면 단독으로는 방향이 있지만, 세 개를 같이 보면 서로 다른 서비스처럼
보인다.

현재 필요한 것은 `화면별 개성 유지`가 아니라 `같은 브랜드의 다른 모드`
처럼 보이게 만드는 연결 규칙이다.

### 3-2. 공개 티켓은 대비가 약하다

특히 공개 티켓 허브와 데스크톱 웹에서 문제가 크다.

- 카드 배경과 버튼 색이 너무 가깝다
- 메타 텍스트가 지나치게 흐리다
- 상태 배지와 CTA가 충분히 튀지 않는다
- 클릭 가능한 영역이 눌리는 버튼보다 어두운 패널처럼 보인다

지금 스크린샷은 `프리미엄`보다는 `어두운 오버레이 위에 묻힌 상태`로
읽힌다.

### 3-3. 공개 티켓 데스크톱 웹이 자랑컷이 아니다

현재 데스크톱 웹은 모바일 캔버스를 중앙에 억지로 세워둔 느낌이 강하다.

- 주변 배경이 지나치게 어둡다
- 카드 폭은 좁고 세로 길이는 길어 답답하다
- 배경과 카드의 분리가 약해 피사체가 떠오르지 않는다
- 첫 화면을 캡처해도 보여주고 싶은 화면보다 테스트 화면처럼 보인다

### 3-4. 인터랙션 신뢰감이 떨어진다

디자인에서 가장 위험한 지점은 `버튼이 버튼처럼 안 보이는 것`과
`눌렀는데 반응이 없는 것`이다.

현재 그룹 티켓 허브의 `티켓 보기`는 스크린샷 기준으로 가장 큰 문제다.
버튼 시각성도 약한데 실제 무반응까지 겹쳐 `품질이 낮다`는 인상을 준다.

### 3-5. 정보 위계가 중간중간 흔들린다

- 어떤 화면은 공연명이 중심이고
- 어떤 화면은 상태 배지가 중심이고
- 어떤 화면은 CTA가 중심이고
- 어떤 화면은 장식용 영어 라벨이 먼저 눈에 들어온다

브랜드가 고급스럽게 보이려면 `어디서든 가장 먼저 보는 정보`가 일정해야
한다.

## 4. 화면군별 감사

### 4-1. 사용자 앱 메인

대상:
- BOOKING
- DISCOVER
- TICKETS
- PROFILE

평가:
- 기본 구조는 가장 안정적이다.
- 하단 탭, 카드, 헤더는 이미 서비스 형태를 갖췄다.
- 다만 탭/카드/티켓/상세 화면 사이의 컴포넌트 언어가 완전히 통일되지는 않았다.

문제:
- 탭 라벨과 아이콘은 여전히 살짝 유틸리티 앱처럼 보인다.
- 일부 화면은 고급스럽고, 일부 화면은 실용적이라 톤이 오락가락한다.

방향:
- 메인 앱은 `밝은 편집형 럭셔리` 톤을 유지
- 공개 티켓은 `프리미엄 소장물` 톤으로 분기
- 하지만 버튼 모서리, 라벨 무드, 카드 깊이는 공통 규칙으로 묶기

### 4-2. 공연 상세 / 결제

평가:
- 공연 상세는 편집형 레이아웃으로 충분히 좋은 편이다.
- 결제 화면도 정보 정리는 좋다.

문제:
- 공연 상세는 영문 섹션 타이틀과 한글 정보가 섞이면서 톤이 조금 분산된다.
- 결제는 안전하고 깔끔하지만 아직 `프리미엄 구매 경험`보다는
  `정돈된 폼`에 가깝다.

방향:
- 공연 상세는 영어 타이틀을 줄이거나 역할을 명확히 제한
- 결제는 CTA, 요약 카드, 결제 수단 선택에서 더 명확한 위계 필요

### 4-3. 내 티켓 / 티켓 상세

평가:
- 메인 앱 내 티켓 화면은 실용성과 브랜드 균형이 괜찮다.
- 티켓 상세도 정보량은 충분하다.

문제:
- 공개 티켓과 일반 티켓의 무드가 이어지지 않는다.
- 같은 서비스인데 `내 티켓`은 밝고, `공개 티켓`은 완전히 다른 서비스처럼
  느껴진다.

방향:
- 일반 티켓은 실용성을 유지하되
- 공개 티켓과 공유하는 공통 모듈
  예: 상태 배지, 타이포 위계, 좌석 정보 표기, QR 강조 방식을 맞출 것

### 4-4. 공개 모바일 티켓 / 그룹 허브

평가:
- 가장 잠재력이 큰 화면이다.
- 동시에 현재 가장 위험한 화면이다.

문제:
- 대비 부족
- CTA 무반응 체감
- 데스크톱에서 지나치게 어두움
- 장식과 정보의 비중이 뒤섞임
- `자랑하고 싶은 화면`보다 `미완성 컨셉 화면`처럼 느껴짐

우선 개선 포인트:
- 이름
- 공연명
- 상태
- 좌석 공개 정보
- CTA

이 다섯 가지를 먼저 선명하게 만들고, 장식은 그 다음이다.

### 4-5. 로그인

평가:
- 브랜드 방향은 있다.
- 소셜 로그인과 체험모드 등 기능은 충분하다.

문제:
- 로그인 화면은 현재 정보량이 많고, 브랜드 메시지보다 기능 모음처럼 보일 수 있다.
- 앱 메인과 티켓의 프리미엄 무드 사이에서 살짝 중간 지점에 머문다.

방향:
- 로그인은 `안심감 + 프리미엄 입장 게이트`처럼 보여야 한다.
- 버튼 수가 많을수록 시선 우선순위를 더 강하게 정해야 한다.

### 4-6. 어드민

평가:
- 툴로서는 충분히 강하다.
- `dark + gold` 방향도 나쁘지 않다.

문제:
- 사용자 앱과 너무 다른 제품처럼 보인다.
- 운영 도구라 해도 브랜드 연결고리가 부족하다.

방향:
- 어드민은 현재 다크 모드를 유지하되
- 컬러, 타이포, 라벨 톤에서 사용자 앱과 연결점을 만들 것

## 5. 현재 스크린샷 기준 직접 문제

### P0

- 그룹 티켓 허브의 `티켓 보기`가 눌러도 반응하지 않는 것처럼 보임
- 그룹 허브 CTA와 메타 텍스트 대비가 너무 약함
- 데스크톱 웹 첫 화면이 지나치게 어둡고 답답함

### P1

- 공개 티켓 허브와 개별 티켓의 스타일 연결이 덜 정리됨
- 메인 앱과 공개 티켓 사이의 브랜드 연속성이 약함
- 공개 티켓의 영어 장식 라벨이 고급감보다 혼선을 줄 가능성 있음

### P2

- 어드민까지 포함한 브랜드 통일 규칙 문서화 부족
- 공유용 자랑컷 기준이 아직 고정되지 않음

## 6. `자랑하고 싶은 화면` 기준

앞으로 공개 티켓은 아래 기준을 만족해야 한다.

- 첫 화면 캡처만으로도 공연 이름과 소유자가 읽힌다
- CTA가 선명하고 눌러보고 싶다
- 배경은 분위기를 만들지만 정보를 잡아먹지 않는다
- 모바일과 데스크톱 모두 `작품`처럼 보인다
- 카카오톡, 인스타, 브라우저 열람 모두에서 같은 브랜드로 인식된다

### 2단계

- MT-077 공개 모바일 티켓 비주얼 시스템 고급화

### 3단계

- MT-079 공개 티켓 데스크톱 웹 레이아웃과 자랑용 연출 마감

- MT-078 멜론티켓 전체 브랜드 일관성 재정비

## 8. 결론

멜론티켓은 지금 `촌스럽다`거나 `기본기가 없다`가 아니라,
`좋은 방향을 여러 개 잡아놓고 아직 하나의 브랜드로 완성하지 못한 상태`다.

따라서 다음 작업은 새 화면을 더 만드는 것이 아니라,

- 눌러야 할 것이 분명하게 보이게 하고
- 읽어야 할 것이 선명하게 읽히게 하고
- 서비스 전체가 하나의 취향과 품질로 보이게 만드는 것

에 집중해야 한다.

그렇게만 정리되면 지금 공개 티켓은 충분히
`보여주고 싶은 티켓`,
`인스타에 올리고 싶은 티켓`,
`카톡으로 보내도 자랑이 되는 티켓`
쪽으로 갈 수 있다.
