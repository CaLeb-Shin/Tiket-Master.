## 2. 타이포그래피 위계

### 2-1. 폰트 스택

| 폰트 | 역할 | 메서드 |
|-------|------|--------|
| NanumBarunGothic | 한국어 본문, 라벨, CTA | `AppTheme.nanum()` |
| Noto Serif | 제목, 이름, 프리미엄 텍스트 | `AppTheme.serif()` |
| RobotoMono | 카운트다운, QR 토큰 (기능적 모노스페이스) | `GoogleFonts.robotoMono()` |

### 2-2. 크기/용도 기준

| 용도 | 크기 | 무게 | 폰트 |
|------|------|------|------|
| 영웅 타이틀 | 30-42px | w900 | nanum |
| 이름 (소유자) | 24-30px | w400 | serif |
| 섹션 타이틀 | 18-20px | w800-w900 | nanum |
| 본문 | 14-15px | w600-w800 | nanum |
| 라벨 | 11-12px | w600-w700 | nanum |
| 메타 힌트 | 10-11px | w600 | nanum |
| 카운트다운/토큰 | 13px | w700 | robotoMono |

### 2-3. 그림자 규칙

- **크림 배경 카드 위**: `noShadow: true` 필수
- **버건디 배경 위**: 기본 그림자 사용 (nanum/serif 기본값)
- **가격/강조 텍스트**: `textShadowStrong` 사용

## 3. 톤 연결 규칙

### 3-1. 메인 앱 ↔ 공개 티켓

| 요소 | 메인 앱 | 공개 티켓 | 연결 포인트 |
|------|---------|-----------|-------------|
| 배경 | 크림 `#FAF8F5` | 버건디 `#3B0D11` → `#1A0508` | 같은 버건디가 메인 앱에서는 텍스트로, 티켓에서는 배경으로 사용 |
| 카드 | 화이트 | 크림 `#FAF8F5` | 메인 앱 배경색이 티켓 카드색으로 순환 |
| 활성 강조 | 버건디 `#3B0D11` | 골드 `#C9A84C` | 역할 반전: 밝은 배경에서는 버건디, 어두운 배경에서는 골드 |
| 이름 표시 | serif, 버건디 | serif, 크림 or 버건디 | 같은 serif 폰트, 배경에 따라 색상만 반전 |

### 3-2. 메인 앱 ↔ 어드민

| 요소 | 메인 앱 | 어드민 | 연결 포인트 |
|------|---------|-------|-------------|
| 브랜드 색 | 버건디 `#3B0D11` | 웜 골드 `#FFCF99` | 같은 웜 톤 계열 (브라운/골드 축) |
| 배경 | 크림 (밝음) | 차콜 (어두움) | 밝기만 반전, 톤은 웜 유지 |
| 폰트 | 같은 스택 | 같은 스택 | NanumBarunGothic + Noto Serif + Inter |

## 4. 컴포넌트 규칙

### 4-1. CTA 버튼

- **메인 앱**: 버건디 배경 + 크림 텍스트
- **공개 티켓 (어두운 배경)**: 골드 배경 + 버건디 텍스트
- **비활성**: 알파 50% 이하, 보더만 표시
- **최소 높이**: 46px (터치 영역)
- **라운드**: 12-14px

### 4-2. 상태 배지

- 사용 가능: 골드 `#C9A84C`
- 공개 전: 골드 `#C9A84C`
- 입장 완료: 초록 `#58C27D`
- 사용 완료: 초록 `#58C27D`
- 취소됨: 빨강 `#FF7A7A`

### 4-3. 카드 패널

- **밝은 배경 위**: 화이트, 0.5px 보더 `AppTheme.border`
- **어두운 배경 위**: 10% 화이트, 14% 화이트 보더
- **라운드**: 16-24px
- **패딩**: 16-20px

### 4-4. 하단 네비게이션

- **활성 아이콘**: `AppTheme.gold` (버건디)
- **비활성 아이콘**: `AppTheme.sage` (세이지)
- **라벨**: 9px, Inter, 2.0 letter-spacing
- **탭**: BOOKING, DISCOVER, TICKETS, PROFILE

## 6. 아이콘 사용 기준

- **Material Rounded** 계열만 사용 (`Icons.*_rounded`)
- Outlined 변형은 비활성 탭에서만 사용
- 크기: 14px (메타), 16px (인라인), 20px (액션), 22px (네비게이션)
